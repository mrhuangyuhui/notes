## Install

<https://router.vuejs.org/en/installation.html>

### CDN

```html
<script src="https://unpkg.com/vue"></script>
<script src="https://unpkg.com/vue-router"></script>
```

### NPM

```bash
npm install vue-router
```

```js
import Vue from 'vue'
import VueRouter from 'vue-router'

Vue.use(VueRouter)
```
