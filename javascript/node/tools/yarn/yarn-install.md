# Yarn Install

- [2.x](#2x)
- [1.x](#1x)

## 2.x

<https://yarnpkg.com/getting-started/install>

Global Install

```bash
npm install -g yarn
```

Per-project install

```bash
cd ~/path/to/project

# below v1.22
yarn policies set-version berry

# on v1.22+
yarn set version berry
```

验证安装结果

```bash
yarn --version
```
