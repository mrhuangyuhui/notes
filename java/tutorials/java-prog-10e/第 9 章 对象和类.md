## 9.4 使用构造方法构造对象

> 一个类可以不定义构造方法。在这种情况下，类中隐含定义一个方法体为空的无参构造方法。这个构造方法称为默认构造方法（default constructor），当且仅当类中没有明确定义任何构造方法时才会自动提供它。

## 9.12 不可变对象和类

> 要使一个类成为不可变的，它必须满足下面的要求：
> - 所有数据域都是私有的。
> - 没有修改器方法。
> - 没有一个返回指向可变数据域的引用的访问器方法。
