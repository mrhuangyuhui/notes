## Commands

查看帮助

```bash
bundle -h
bundle --help

# 查看子命令的帮助
bundle help install
```

### `bundle config`

```bash
bundle help config
```

列出所有配置项

```bash
bundle config list
```

配置 Gemfile 的源镜像

```bash
bundle config mirror.https://rubygems.org https://gems.ruby-china.com
```
